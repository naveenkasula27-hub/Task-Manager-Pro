# 🗂️ Task Manager Pro

A simple and interactive task management web application built using **HTML, CSS, and JavaScript**.  
This project helps users add, manage, and track their daily tasks easily in the browser.

---

## 🚀 Live Demo

👉
https://naveenkasula27.ccbp.tech/

## 📌 Features

- Add new tasks dynamically
- Mark tasks as completed (checkbox toggle)
- Delete tasks individually
- Save tasks using browser local storage
- Data persists even after page refresh
- Clean and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 4 (for layout)
- Local Storage API

## ⚙️ How to Run Locally

1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html` in any browser
4. Start adding your tasks

---

## 🌐 How to Deploy (GitHub Pages)

1. Push code to GitHub repository
2. Go to **Settings → Pages**
3. Select:
   - Branch: `main`
   - Folder: `/root`
4. Save
5. Your live link will be generated automatically
